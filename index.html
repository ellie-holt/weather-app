This is a weather app created as part of the SheCodes Plus course. It is far from perfect, but building this app has taught me a lot, particularly about using APIs and CSS frameworks like Bootstrap. Although it was not required for the course, I included some extra features such as weather-depended colour themes and some extra unit conversion. As I learn more, I am keen to update this app by extended the temperature conversion function to the forecast and incorperating accurate times and dates for different time zones.

For now, however, I must press submit! Thanks for checking out this project!
